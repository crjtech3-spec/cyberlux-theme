Quick start (most common tasks)
- Change logo: `Theme settings > Favicon & Logo > Logo`
- Change colors: `Theme settings > Colors` (pick a preset or `Custom`)
- Add a homepage section: `Customize theme > Add section` and choose the section
- Reorder homepage blocks: drag-and-drop in the Theme Editor sidebar
