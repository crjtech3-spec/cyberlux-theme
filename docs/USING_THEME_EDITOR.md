# Using the Theme Editor

## Overview
Cyber Lux exposes a large set of homepage sections and site-wide settings. Use the Shopify Theme Editor (Customize) to:
- Add, remove or reorder homepage sections
- Switch presets (Design > Design preset)
- Configure header, footer, product and collection settings

## Common tasks
- Change logo: `Theme settings > Favicon & Logo > Logo`
- Swap color palette: `Theme settings > Colors > Theme color palette`
- Edit typography: `Theme settings > Typography`
- Add a homepage block: `Sections > Add section > choose section`

## Presets
To preview a preset, open `listings/<preset>/index.*.json` in the Theme Editor (or select preset from demo importer if available).
