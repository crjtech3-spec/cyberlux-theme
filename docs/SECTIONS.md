# Sections reference

Cyber Lux includes 84 section files. Below are the primary homepage-eligible and page-specific sections with short descriptions.

## Homepage-eligible sections (selected)
- `image-banner` — hero image with heading, text and CTA.
- `slideshow` — full-width slides with images or video.
- `featured-collection` — curated product grid.
- `featured-product` — single product highlight block.
- `product-carousel` — carousel of product cards.
- `logo-carousel` — brand logos / trust marks.
- `rich-text` / `rich-story` — editorial content with images.
- `faq-accordion` — FAQ with expandable items.
- `comparison-table` — product feature comparison table.
- `shop-the-look` / `lookbook` — curated outfits / shoppable imagery.
- `video-hero` / `video-gallery` — native video sections.
- `newsletter` — email capture
- `product-recommendations` / `recently-viewed` — cross-sell & personalization
- `promo-popup` — opt-in / announcement pop-up
- `complete-your-look` / `frequently-bought-together` — upsell sections

## Page-specific (main-*)
- `main-product` — product page layout and gallery
- `main-collection-product-grid` — collection page
- `main-search` — search results layout
- `main-article` / `main-blog` — article/blog templates
- `main-cart-items` / `main-cart-footer` — cart page templates

(Full list is included in the repository `sections/` folder.)

---

Tip: Shopify Theme Store requires 25+ homepage-eligible sections — Cyber Lux provides 59 homepage-eligible sections (well above the requirement).
